# Love Note

Love Note is a cozy, romantic app for sharing mood updates, special dates, and love notes with a partner. It includes partner linking, photo frame uploads, reminders for upcoming anniversaries, and a sweet in-app journal experience.

## Features

- Link accounts with your partner for shared moods and connected experiences
- Upload and crop a favorite photo for a personalized profile frame
- Track upcoming dates and anniversaries in a romantic dashboard
- Save quick notes locally and sync them with the cloud
- Toggle themes and manage account settings from the dashboard

## Local Development

```bash
npm install
npm run dev
```

Open the app in your browser at `http://localhost:5173`.

## Build

```bash
npm run build
```

## Notes

- The app uses Supabase for authentication, storage, and realtime data.
- Public social preview assets are pulled from `/logo.png`.
- For full partner linking and account deletion, configure the backend API and auth environment variables.
